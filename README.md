# 가구 & 강아지 용품 신상 홍보 페이지

2026년 봄 신상 가구와 강아지 용품을 홍보하기 위한 모던한 랜딩 페이지입니다.

## 🌟 주요 기능

- **반응형 디자인**: 모든 기기에서 완벽하게 보입니다
- **모던 UI**: TailwindCSS를 사용한 세련된 디자인
- **인터랙티브 요소**: 부드러운 애니메이션과 효과
- **제품 카드**: 가구와 강아지 용품을 아름답게 표시
- **프로모션 섹션**: 특별 할인 혜택 안내
- **문의 양식**: 고객 문의를 받을 수 있는 폼

## 🛠️ 사용된 기술

- **HTML5**: 시맨틱 마크업
- **TailwindCSS**: CSS 프레임워크
- **JavaScript**: 인터랙티브 기능
- **Font Awesome**: 아이콘 라이브러리
- **Google Fonts**: 한글 폰트 (Noto Sans KR)

## 📱 페이지 구조

1. **네비게이션**: 고정 헤더와 모바일 메뉴
2. **히어로 섹션**: 메인 비주얼과 CTA 버튼
3. **가구 섹션**: 신상 가구 제품 카드
4. **강아지 용품 섹션**: 반려견 용품 카드
5. **프로모션 섹션**: 특별 할인 혜택
6. **문의 섹션**: 연락처와 문의 양식
7. **푸터**: 회사 정보와 링크

## 🎨 디자인 특징

- **그라데이션 효과**: 보라색과 핑크색 톤
- **카드 호버 효과**: 마우스 오버시 애니메이션
- **페이드인 애니메이션**: 스크롤시 나타나는 효과
- **모바일 최적화**: 반응형 그리드 시스템

## 🚀 실행 방법

1. 프로젝트 폴더로 이동:
   ```bash
   cd furniture-dog-promo
   ```

2. 브라우저에서 `index.html` 파일 열기:
   ```bash
   open index.html
   ```

   또는 로컬 서버로 실행:
   ```bash
   python3 -m http.server 8000
   # http://localhost:8000 에서 접속
   ```

## 📝 커스터마이징

- **색상 변경**: CSS 변수 또는 TailwindCSS 클래스 수정
- **이미지 교체**: `<img>` 태그의 `src` 속성 변경
- **텍스트 수정**: HTML 파일의 텍스트 내용 직접 편집
- **제품 추가**: 제품 카드 구조 복사하여 추가
